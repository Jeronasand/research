主要端点清单

以下不是逐字复制 OpenAPI，而是面向调研和架构设计整理的“能力索引”。具体字段、路径细节和 scope 以官方 API Reference 为准。

模块端点 / 入口名称能力备注
Executions and ResultsPOST /api/v1/sql/executeExecute SQL直接执行原始 SQL，无需先保存 Query。适合动态分析、Agent 临时查询。支持 performance: small / medium / large。返回 execution_id。
Executions and ResultsPOST /api/v1/query/{query_id}/executeExecute Query执行已保存 Query。支持参数化 Query。适合生产化调用：先在 Dune UI 调好 SQL，再由 API 执行。
Executions and ResultsGET /api/v1/execution/{execution_id}/statusGet Execution Status查询执行状态。用于轮询 pending/running/completed/failed。
Executions and ResultsGET /api/v1/execution/{execution_id}/resultsGet Execution Result按 execution_id 获取结果。适合拿到刚执行任务的结果。
Executions and ResultsGET /api/v1/query/{query_id}/resultsGet Latest Query Result获取某个 Query 最近一次执行结果。不会触发执行，但按结果大小消耗 credits。支持分页/过滤/排序/采样。
Executions and ResultsGET /api/v1/query/{query_id}/results/csvGet Latest Query Result CSVCSV 格式获取最近结果。适合导出、离线分析、报表下载。
Executions and ResultsPOST /api/v1/execution/{execution_id}/cancelCancel Execution取消执行任务。用于超时控制和 Agent 自我保护。
Query ManagementPOST /api/v1/queryCreate Query创建 Query。可由 Agent/系统自动生成查询资产。
Query ManagementGET /api/v1/query/{query_id}Read Query读取 SQL、参数、名称、标签、状态。私有 Query 需拥有者 API Key。
Query ManagementPATCH /api/v1/query/{query_id}Update Query更新 SQL、参数、名称、标签、状态。适合自动维护模板 Query。
Query ManagementGET /api/v1/queriesList Queries列出当前 API Key 所属账号的 Query。适合资产索引。
Query ManagementPOST /api/v1/query/{query_id}/archiveArchive Query归档 Query。不会删除，但变为不可编辑/不可执行。
DatasetsGET /api/v1/datasetsList Datasets分页列出数据集，可过滤 owner/type。Agent 数据发现入口。
DatasetsGET /api/v1/datasets/searchSearch Datasets按分类、链、schema 等搜索数据集。用于自然语言找表。
DatasetsGET /api/v1/datasets/{id}Get Dataset获取数据集元数据、字段、所有者等。用于生成 SQL 前的 schema inspection。
DatasetsGET /api/v1/datasets/search-by-contractSearch by Contract Address通过合约地址查 decoded datasets。适合合约事件分析。
UploadsPOST /api/v1/uploads/csvUpload CSV上传 CSV 并自动推断 schema，生成可查询表。单次上传限制 500MB；不同 plan 有存储上限。
UploadsPOST /api/v1/uploads/createCreate Table创建空表，后续通过 insert 写入。每次建表消耗 credits。
UploadsPOST /api/v1/uploads/insertInsert Data向已有表插入 CSV 或 NDJSON。适合把内部策略数据、钱包标签、监控结果写入 Dune。
UploadsPOST /api/v1/uploads/clearClear Data清空上传表数据，保留结构。用于刷新外部数据快照。
UploadsDELETE /api/v1/uploads/{table}Delete Table删除上传表及数据。高风险操作，Agent 应禁用或需要人工确认。
Materialized ViewsPOST /api/v1/materialized-viewsUpsert Materialized View从现有 Query 创建/更新物化视图。用于降低重复计算。
Materialized ViewsPOST /api/v1/materialized-views/{name}/refreshRefresh Materialized View刷新物化视图。默认 medium，可传 performance: large。
Materialized ViewsGET /api/v1/materialized-viewsList Materialized Views列出账号拥有的物化视图。适合管控数据管道资产。
PipelinesPOST /api/v1/pipelines/{id}/executeExecute Pipeline执行一个由多个 Query/物化视图刷新组成的 pipeline。适合复杂依赖任务。
PipelinesGET /api/v1/pipelines/executions/{id}/statusGet Pipeline Status查看 pipeline 内每个节点的状态。用于调度系统。
UsageGET /api/v1/usageGet Usage查询用量与 billing period。Agent 成本控制必接。
WebhooksUI Schedule WebhookCreate WebhookQuery schedule 之后把结果推送到 webhook URL。适合 Slack、通知系统、AI Pulse。

Agent 安全建议

  • 默认只允许 Agent 读取、执行有限 Query，不允许删除表、删除物化视图、归档 Query。
  • 所有动态 SQL 必须加时间范围、LIMIT、目标链、目标表白名单。
  • 大查询走 Materialized View，不要每次重新扫链。
  • 所有 API Key 不要写入前端。
