01

核心结论

适合用作 DeFi 数据源、链上指标看板、研究自动化、AI Agent 数据工具和内部监控。

API 能力

核心公开 API 覆盖协议 TVL、链 TVL、价格、收益池、稳定币、费用/收入等常用数据；Pro/API 计划提供更高限额和更多高级数据类型。

定价结构

当前可拆成 Free、Pro、API、Enterprise。Pro 偏研究产品能力；API 计划偏开发者、高频调用、MCP 和高级端点。

Web 产品

Web 端不仅是数据看板，还包含 LlamaAI、CSV Downloads、Custom Dashboards、Sheets、LlamaFeed、Alert 等研究工作流。

接入重点

前端可直连公开端点做 Demo；生产环境建议后端代理、缓存、限流重试、API Key 隔离和数据快照。

03

如何使用

前端可直接 fetch 公开端点；生产环境建议后端代理，避免把 API Key 暴露给浏览器。

1 确定数据域

TVL、价格、收益池、稳定币、费用/收入、RWA 或下载数据集。

2 选择端点

从公开 API 开始；需要高级数据或高频调用时切换 API 计划。

3 缓存与归档

按分钟/小时缓存，保存快照，减少重复请求和限流风险。

4 业务展示

将 JSON 转为表格、趋势图、告警规则或 AI Agent 上下文。

示例 A：获取全部协议并筛选 Aave 
async function getAaveProtocol() {
  const res = await fetch('https://api.llama.fi/protocols');
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const protocols = await res.json();

  return protocols.find((item) => item.slug === 'aave');
}
示例 B：获取协议详情与历史 TVL 
async function getProtocolDetail(slug = 'aave') {
  const url = `https://api.llama.fi/protocol/${encodeURIComponent(slug)}`;
  const res = await fetch(url);
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  return res.json();
}
示例 C：获取 USDC 当前价格 
async function getUsdcPrice() {
  const coin = 'ethereum:0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48';
  const res = await fetch(`https://coins.llama.fi/prices/current/${coin}`);
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  return res.json();
}
示例 D：带退避的请求封装 
async function llamaFetch(url, retries = 2) {
  for (let i = 0; i <= retries; i += 1) {
    const res = await fetch(url);
    if (res.ok) return res.json();
    if (res.status !== 429 || i === retries) throw new Error(`HTTP ${res.status}`);
    await new Promise((resolve) => setTimeout(resolve, 600 * 2 ** i));
  }
}

06

LlamaAI 与 Schedule Alert

截图显示 LlamaAI 能将自然语言转成研究图表、解释、来源引用和可保存的告警任务。

LlamaAI 能力拆解

  • 自然语言查询：例如“计算 R25 协议 TVL 变化与流入趋势”。
  • 数据计算：读取协议 TVL、变动百分比、时间窗口和链/分类上下文。
  • 可视化：生成趋势图和状态表，并提供 PNG/CSV。
  • 来源归因：回答中列出 DefiLlama 数据和外部来源链接。
  • 后续行动：自动给出 follow-up 按钮，例如同类对比、链上追踪、增长排行。

Schedule Alert 能力拆解

  • 触发条件：R25 TVL 在 24 小时内变化超过 ±10%。
  • 调度频率：Daily，截图中配置为 UTC+8 11:00。
  • 投递方式：Email 到注册邮箱；支持将 LlamaAI 账号关联 Telegram Bot。
  • 保存动作：生成配置后仍需用户点击 Save Alert 确认。
  • 权限边界：截图证明可生成配置页，但是否长期可用、是否收费，需以账号权限和保存结果为准。

Web 产品与 API 的关系

Web 端是研究工作台；API 是数据接口；MCP 是给 AI Agent 使用的数据工具层。

DefiLlama 数据层
Adapters / DB / Market datasets
→
API / Pro API / MCP
JSON、OpenAPI、Agent Tools
→
Web Dashboard / LlamaAI
图表、研究、导出、提醒
→
用户工作流
投研、监控、报表、自动化

公开 API

适合快速拉取协议、链、TVL、价格等数据，构建 MVP、内部看板或低频监控。

API 订阅

适合商业化产品、AI Agent、数据密集型后台、需要更高限额或高级端点的场景。

LlamaAI

适合研究员直接提问，快速得到计算、图表、来源和可保存的告警配置。
