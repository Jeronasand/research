# Dune 中文调研包

本目录是 Dune 数据源提供商调研的离线 HTML 版本。

## 打开方式

直接打开 `index.html` 即可。

## 内容

- API 能力
- 主要端点清单
- MCP / AI Agent
- Dune CLI & Skills
- DuneSQL 查询引擎
- 数据目录
- 上传数据
- Materialized Views
- Pipelines / Webhooks
- SDK / Connectors
- 计费 / 限制
- 使用场景
- 集成架构
- 供应商评分

## 数据文件

`data/endpoints.json` 可作为后续索引或渲染数据源。
