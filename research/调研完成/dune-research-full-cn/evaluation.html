供应商评分

按数据源提供商调研模板对 Dune 做初步评分。

维度评分说明
链上数据覆盖5/5覆盖多链、raw/decoded/curated/community 数据。
传统金融覆盖0/5不是股票、外汇、宏观数据源。
API 完整度5/5执行、结果、Query 管理、上传、MV、Pipeline、Usage。
AI Agent 兼容5/5官方 MCP、CLI、Agent Skills。
实时性2/5适合分析和定时监控，不适合 tick/orderbook。
成本可控性3/5Credit 模型灵活，但 Agent 必须做限制。
开发体验4/5文档完整，支持 SDK 和 CLI；需要熟悉 SQL/链上表。
企业化4/5有 Trino、dbt、Datashare、BI 方向能力。

最终结论

  • 推荐等级：强烈推荐用于 Web3 链上数据与 AI Agent 数据源。
  • 不推荐：作为低延迟交易行情源、股票源、盘口源。
  • 最佳接入方式：MCP 用于探索；API + Materialized Views + Webhook 用于生产。
  • 最佳本地架构：Go 服务代理 Dune API，Codex/AI Pulse 调用本地服务，而不是直接暴露 Dune API Key。
