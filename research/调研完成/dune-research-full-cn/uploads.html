Data Uploads：上传自定义数据

Dune 支持把自己的数据上传成可查询表。这个能力对 AI Agent 非常关键，因为它可以把内部数据、策略结果、钱包标签与链上数据放到同一个 SQL 环境里 Join。

两种上传方式

方式说明优点缺点
CSV UploadUI 或 API 上传 CSV，自动推断 schema。简单，适合快速验证。schema 控制弱，不适合复杂增量数据。
Create Table + Insert先创建表，再插入 CSV 或 NDJSON。更适合程序化、增量、Agent 工作流。需要提前设计 schema 和写入流程。

可上传的数据类型建议

  • 钱包标签：地址、标签、风险等级、来源。
  • 交易策略结果：signal_id、token、score、reason、created_at。
  • 内部监控事件：alert_type、chain、contract、value、timestamp。
  • 用户自定义 watchlist：wallet、token、protocol。
  • 外部行情快照：token、price、volume、source。

示例 schema：钱包标签表

team.wallet_labels
- address varbinary / varchar
- label varchar
- category varchar
- risk_level varchar
- source varchar
- updated_at timestamp
