Dune CLI & Agent Skills

Dune 提供 CLI 和 Agent Skills。CLI 是终端原生工具；Skill 是给 AI Coding Agent 的能力包，教 Agent 如何用 CLI 发现数据集、写 DuneSQL、执行查询、处理错误和控制成本。

安装

curl -sSfL https://dune.com/cli/install.sh | sh

官方说明该脚本会安装 Dune CLI、运行 dune auth 配置 API Key，并通过 npx skills add 安装 Dune Agent Skill。

认证

# 交互式认证
dune auth

# 环境变量
export DUNE_API_KEY=<your-api-key>

# Inline API Key
dune query run 12345 --api-key <your-api-key>

机器可读输出

CLI 命令支持 -o json，适合 Agent 调用。

dune query run-sql --sql "SELECT * FROM ethereum.transactions LIMIT 5" -o json

对 SkillGenome 的价值

  • 可以把 Dune 调研和最佳实践变成一个 Skill HTML 页面。
  • Skill 内容包括：什么时候用 Dune、怎么找表、怎么写 SQL、怎么限制成本、怎么把结果推给 AI Pulse。
  • 可以将常见链上分析任务封装成技能模板：钱包分析、DEX 交易量、CEX 流入流出、鲸鱼监控、NFT floor、Gas 异常。
