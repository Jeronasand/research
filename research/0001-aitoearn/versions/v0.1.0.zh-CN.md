## 范围

AiToEarn 首版公开来源调研快照，基于 2026-05-11 访问的中国站、国际站、官方文档、GitHub 仓库、更新日志、定价文档和状态页。

## 结论

AiToEarn 是一个开源、AI 驱动的社交媒体与内容变现平台。它把创作者推广任务市场与发布、互动、创作、数据分析流程组合在一起。

中国公开站点突出创作者通过推广任务赚钱。观察到的任务包括置顶即梦 CPE 推广任务、小红书 CPE 任务、抖音相关历史任务，以及低单价互动任务。国际站使用美元展示平行任务市场，也暴露创作者和广告主两类角色。

官方产品叙事围绕四个支柱：Monetize、Publish、Engage、Create。Monetize 覆盖任务式创作者收益和广告主推广任务；Publish 覆盖多平台排程与分发；Engage 覆盖 AI 辅助回复、评论搜索和社交互动；Create 覆盖 AI 文案、图片和视频内容生成。

公开文档和 GitHub 显示出较强开放技术姿态：MIT license 仓库、Docker Compose 部署、源码开发路径、MCP/SSE 端点、OpenClaw 插件和 API Key 集成。自部署看起来可行，但依赖 OAuth/渠道配置或 relay 使用。

发布和运维证据存在混杂。文档更新日志显示 2026-04-29 的 v2.4.0，但公开观察到的 GitHub 仓库 Releases 面板显示 2026-03-28 的 v2.1.0 为 latest。状态页显示活跃问题和很低 uptime，因此 SaaS 可靠性需要直接验证。

定价文档显示 Free/Plus 计划和积分定价，但定价页面日期停留在 2025-11。除非在应用内或当前计费页面重新核对，否则应把定价视为可能过期。
