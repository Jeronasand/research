## 关键结论

- AiToEarn 将自己定位为面向创作者、品牌、小团队和一人公司的开源 AI 社交媒体与内容变现平台。
- 它不是单纯的推广任务列表。公开材料展示的核心能力包括 Monetize、Publish、Engage、Create 四个方向。
- 中国站更突出创作者接推广任务赚钱；国际站使用美元结算展示类似任务模型，并强调全球社交渠道。
- 开源信号较强：公开 GitHub 仓库、MIT license、Docker Compose 自部署、MCP/SSE 接入点、桌面客户端相关说明。
- 文档和发布渠道没有完全对齐：帮助文档更新日志显示 2026-04-29 的 v2.4.0，但 GitHub 仓库页面观察到的 Releases 面板显示 2026-03-28 的 v2.1.0 为 latest。
- 运维状态需要继续验证：官方状态页当前公开视图显示存在活跃问题，并展示多个产品的极低 uptime 数值。
