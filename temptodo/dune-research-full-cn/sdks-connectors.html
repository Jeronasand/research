SDK / Connectors

Dune 不只提供 HTTP API，也提供 SDK、Trino Connector、dbt Connector，以及面向 BI 工具的连接方式。

SDK

SDK用途建议
Python SDK研究脚本、数据分析、Notebook、调度任务。适合快速验证和量化研究。
TypeScript / JavaScript SDKWeb 后端、Node 服务、Agent 工具。适合你现有 Web3 / 全栈项目。
Go SDK后端服务、定时任务、AI Pulse 服务。适合你当前 Go 后端方向。

Trino Connector

适合企业把 Dune 当作远程数据仓库，用 Trino/Presto 生态工具直接连接，例如 DBeaver、Metabase、Hex 或自有 BI 系统。

  • DBeaver：数据库客户端式查询。
  • Metabase：BI Dashboard。
  • Hex：Notebook / 分析工作流。
  • Trino SDKs：Java/Python/Go 等生态。

dbt Connector

适合把 Dune 数据变成正式的数据工程管道，支持 dbt project、incremental models、CI/CD 和数据转换。

你的项目建议

  • Go 服务：优先 HTTP API / Go SDK。
  • Codex：优先 MCP + Skill。
  • Web 管理后台：直接调用你的后端，不要前端直连 Dune。
  • 复杂数据工程：再考虑 dbt / Trino Connector。
