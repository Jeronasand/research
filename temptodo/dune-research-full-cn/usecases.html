使用场景评估

下面按你的 AI Agent / Web3 / Trading Bot / AI Pulse 方向做场景拆解。

场景评分

场景适合度说明
链上研究 Agent★★★★★数据发现 + SQL + 结果解释，非常适合。
钱包画像★★★★★结合 labels、dex、cex flows、balances、NFT 等数据。
鲸鱼监控★★★★☆适合小时级/日级监控；不适合毫秒级。
DeFi 指标★★★★★DEX、lending、bridge、gas、protocol events 数据丰富。
Trading Signal★★★☆☆适合作为慢变量/链上因子，不适合作为低延迟执行信号。
高频交易★☆☆☆☆Dune 不是交易所级实时行情/盘口源。
股票数据☆☆☆☆☆不是股票行情数据源。
AI Pulse 通知★★★★☆通过 schedule + webhook 推送到本地服务。
SkillGenome★★★★★Dune Skill/CLI/MCP 可变成标准化 Skill 页面。

推荐做的 6 个 Skill

  1. Wallet Analyzer：输入钱包地址，输出资产、交易行为、标签、风险。
  2. Token Flow Monitor：输入 token，监控 DEX/CEX/Bridge 流向。
  3. Whale Alert：检测大额交易和异常流入流出。
  4. DEX Volume Research：分析链/协议/token 的成交量变化。
  5. Protocol Dashboard Builder：输入协议名称，自动查表、写 SQL、建 Dashboard。
  6. Onchain Daily Report：每天生成链上日报并触发 AI Pulse。
